# Standing History

**Live:** [https://davoskanyan.github.io/standing-history/](https://davoskanyan.github.io/standing-history/)

Interactive La Liga 2018/19 standings and match results. Browse the table and results by matchweek, and generate a video recap of the season with [Remotion](https://www.remotion.dev/).

**Stack:** React 19, Vite 7, TypeScript, MUI, Remotion

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) (or the URL shown in the terminal).
